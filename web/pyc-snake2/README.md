# PyC-Snake2

## Overview
PyC-Snake2 is a Python-based implementation of the classic Snake game with enhanced features and modern styling. This project demonstrates object-oriented programming principles in Python while providing an engaging single-player game experience.

## Features
- Classic Snake gameplay with increasing difficulty
- Score tracking and high score system
- Game over detection and restart functionality
- Clean, modular code structure
- Keyboard controls (arrow keys)

## Requirements
- Python 3.6+
- pygame library

## Installation
```bash
cd pyc-snake2
pip install pygame
```

## Usage
Run the game with:
```bash
python main.py
```

Use arrow keys to control the snake. Collect food to grow and increase your score. Avoid hitting walls or yourself.

## Project Structure
```
pyc-snake2/
├── main.py          # Main game loop and entry point
├── snake.py         # Snake class definition
├── food.py          # Food class definition
├── game.py          # Game logic and display
├── constants.py     # Game configuration constants
└── README.md        # This file
```
